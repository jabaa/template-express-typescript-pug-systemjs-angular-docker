# Webtemplate

## Description

This is a template for a web app.

### Backend

Node.js, Express

### Frontend

Angular, Typescript, Pug, Sass, Systemjs

### Other

Docker, git, gulp

## System requirements

* docker >= 17.09.0-ce
* docker-compose >= 1.16.1

## Usage

### Start webapp

* docker-compose up
* Open localhost:3001

### Development

Docker image must be rebuilt on every change outside of src
